**This folder contains notebook templates. Before each analysis, please do the following:**

1. Copy the *notebooks* folder into a *projects/project_name/experiment_name* folder. You can name the projects and experiments the way you want.
2. Edit the analysis configuration sections (at the beginning) of the notebooks for project and experiment-specific analysis.
3. Make sure that 3rd party softwares have been installed on your system and are in your path.
4. Run the notebooks and have fun!
